1. アプリ概要

サーバーからPDFをダウンロード・保存し、アプリ内で閲覧できる Flutter 製ビューアーアプリ。日英 2 言語・ライト／ダーク テーマに対応。キーワード検索・ブックマーク・PDF内リンク（InAppWebView）機能を持つ。

2-4. プレビュー表示カード（ContentPreviewCard）
レイアウト仕様（mainAxisExtent: 300px 固定）
領域仕様
カード全体clipBehavior: antiAlias（サムネイルに角丸適用）
サムネイル領域（Expanded） 保存済み: PdfDocumentViewBuilder → ClipRect → OverflowBox(topCenter, maxHeight:∞) → AspectRatio(1/1.414) → PdfPageView(page:1, dpi:150)
未保存: #F0F0F0（ダーク: #2C2C2C）背景＋picture_as_pdfアイコン(52px)＋"未ダウンロード"テキスト
ダウンロード中オーバーレイ: 黒55%半透明＋円形プログレス(44px)＋%テキスト
情報領域（固定） padding: fromLTRB(8,6,8,8)
Row: [カテゴリバッジ(10px)] [保存済みチェック green.shade600 if downloaded] [Spacer] [削除アイコン if downloaded]
タイトル: 12px bold, maxLines:2, overflow:ellipsis
アクション: ダウンロード中/保存済み/未保存で切り替え（ボタン高さ30px、文字12px）
A4比率表示の仕組み: OverflowBox(maxHeight: double.infinity) でページ全体を展開し、AspectRatio(1/1.414) でA4縦比率に固定、ClipRect で下部を切り取ることで、上部（タイトル・表紙部分）を最大幅で表示する。
